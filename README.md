# Guia de Referência Rápida Lenda de Ghanor [![image](https://img.shields.io/badge/rpg-A_Lenda_de_Ghanor_RPG-blue?style=flat-square)](https://jamboeditora.com.br/produto/a-lenda-de-ghanor-rpg-livro-basico)
Este projeto é um guia para mestres e jogadores de [A Lenda de Ghanor RPG](https://jamboeditora.com.br/produto/tormenta20-edicao-jogo-do-ano/) e tem o objetivo de apresentar de forma resumida e intuitiva regras básicas desse sistema de RPG.

[A Lenda de Ghanor RPG](https://jamboeditora.com.br/produto/tormenta20-edicao-jogo-do-ano/) pertence a [Jambo Editora](https://jamboeditora.com.br/). Todos os direitos são reservados à editora.

## ⚙️ Rodar o projeto
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas: [Git](https://git-scm.com/downloads), [Node.js](https://nodejs.org/en/download/). Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/download).

```
# Clonar o repositório
$ git clone https://github.com/EdersonCR/referenceghanor.git

# Entrar no diretório
$ cd reference-ghanor

# Instalar as dependências
$ npm install

# Iniciar o projeto
$ npm run start
```
